# Roadmap

This folder tracks delivery posture. Keep detailed rationale in the relevant intent folders and link to it from these ledgers.

Canonical files:

- `00-executive-summary.md`: concise project direction and current milestone posture.
- `03-feature-backlog.md`: feature priority source of truth.
- `04-agent-discovered-workstreams.md`: discovery/workstream status and next artifact tracker.
- `05-bug-ledger.md`: single canonical defect and regression ledger.

Do not create a second feature priority source in another file. If a discovery packet or review changes priority posture, update or recommend updates here.
